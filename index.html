🖥

KalOS v2.0 Alpha

Custom x86-64 Bare-Metal Operating System — Built from Scratch

C x86-64 ASM Bare-Metal GRUB Multiboot Ring 0 / Ring 3 QEMU + Real Hardware
KalOS Desktop — File Manager, Kal Text Editor, Task Manager & Shell running simultaneously

KalOS is a fully custom operating system built entirely from scratch — no Linux, no Windows, no POSIX. Every component is hand-written in C and x86-64 Assembly: from the bootloader handoff through GDT/IDT setup, paging, memory management, PCI scanning, hardware drivers, a full desktop GUI, networking, and a multi-filesystem VFS. It boots on QEMU and real x86-64 hardware.

🏗 Kernel & Architecture

  • x86-64 Long Mode, 4-level paging
  • Custom GDT, IDT, TSS
  • Physical & virtual memory manager
  • Preemptive multitasking scheduler
  • Ring 0/3 privilege separation
  • INT 0x80 syscall interface
  • ACPI, APIC, PCI bus scanning

🖥 GUI & Applications

  • VESA/VBE framebuffer at 1366×768×32bpp
  • Multi-window desktop with taskbar
  • File Manager (CRUD, search, disk persist)
  • Kal Text Editor (syntax highlighting)
  • Task Manager (live CPU/RAM/process stats)
  • Media Player, Browser, Image Viewer
  • CKAL Scripting Language + IDE

🔌 Hardware Drivers

  • Intel e1000 & RTL8139 NIC drivers
  • AHCI SATA controller
  • USB HID (xHCI keyboard + mouse)
  • Intel HDA audio driver
  • PS/2 keyboard & mouse
  • ATA/PIO disk driver
  • VGA text + VESA graphics

🌐 Networking & Storage

  • Ethernet / IPv4 / UDP stack
  • DHCP client, DNS resolver
  • Custom KalFS filesystem
  • FAT32 read/write support
  • ext2 filesystem support
  • VFS abstraction layer
  • ATA disk persistence across reboots
